# 로지판 (LogiPan)

통합 물류 관리 시스템 - PC 프로그램 + 웹 작업보고

## 📂 구성

| 파일 | 용도 |
|---|---|
| `index.html` | 작업자용 웹 페이지 (현장 보고, 공지/소통) |
| `LogiPan.py` | 관리자용 PC 프로그램 |
| `version.txt` | PC 프로그램 버전 (이걸 바꿔야 자동 업데이트 트리거됨) |
| `requirements.txt` | PC 프로그램 필요 라이브러리 |

## 🌐 웹 (작업자용)

GitHub Pages에서 호스팅 됨. 브라우저로 접속만 하면 됨.

## 🖥️ PC 프로그램 (관리자용)

자동 업데이트 시스템. 관리자가 코드 수정 후 `version.txt` 숫자를 올리면 모든 작업자 PC가 다음 실행 시 자동으로 받아감.

### 버전 규칙
- `1.0.0` → `1.0.1` : 작은 버그 수정
- `1.0.0` → `1.1.0` : 작은 기능 추가
- `1.0.0` → `2.0.0` : 큰 변경
